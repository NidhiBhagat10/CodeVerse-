Keywords - Reserved words that have some meaning in programming.
Var - Variables are used to store values/data.It's value can be changed.
  • in ES5 version
  • Scope: function
  • var adds it self to the window object. It's not good because it exposes your data.
Const - Constant are also used to store value but it's value can no be changed.
let - It is also variable type and used to store data.
  • in ES6 version
  • Scope: braces
  • let and const does not add itself to window object.
Mainly Java-Script have 2 versions ES5 and ES6. ES5 only have var variable type . ES6 have both let and const type.
Browser context API - This provides window, stack, heap memory.

Stack - Stack is a data-structure, use to store data in a structured manner. Here in Stack, data that enters last comes out first.
Heap Memory - heap memory is used to store variable and data of our program.
Execution context - It is an imaginary container, whenever we exicute a function it will make an imaginary container where it will store it's three things
  • variables
  • functions inside that parent function
  • lexical environmentof that function
Lexical environment -It is like a chart of things that function can access, it holds scope and scope chain. It tells what we can access and what we cannot in that function.
