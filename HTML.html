HTML consists of elements

<h1>My name is Anthony Gonsalves <⁄h1>
In the code above,
  • Anything contained in the boxes '<>' is known as a Tag
  • <h1> is known as an Opening tag
  • 'My name is Anthony Gonsalves' is the Content
  • <⁄h1> is known as the Closing Tag
  • '<h1> My name is Anthony Gonsalves <⁄h1>'- the entire unit togetger is called an HTML Element

HTML also allows us to generate Lists

  • Unordered list are created using the < ul > tag. An unordered list has individual items with a bullet point.
    < ul >
    < li > Athens < ⁄ li >
    < li > Volcanic island < ⁄ li >
    < ⁄ ul >
    Will apppear as
    • Athens
    • Volcanic islands
  • Ordered lists are created using the <ol> tags. Insted of the bullet points- here the content is ordered with numbers
    < ol >
    < li > Athens < ⁄ li >
    < li > Volcanic islands < ⁄ li >
    < ⁄ol>

    1. Athens

  • The < li > tag is an Html element used to create a list item in an ordered (<ol>) or unordered(< ul >) list. "li" stands for "list item".

The <img> tag allows you to add image to your webpage.

  • <img> is a self closing tag.
  • The <img> has an attribute src. src must be set as a URL where the image is stored.
  • Sample syntax for using the <img> tag is as follows.
    • < src="https://cnd.codechef.com/learning/webdev" >
  • Usually an alt attribute is also added to an image tag
    • If an image fails to load - it shows the discription of the image.
    • Screen reading software and search engine also read the description of the image as they cannot 'see' the image
    • Sample syntax of using the alt attribute
      • < img src="https://cdn.codechef.com/Learning/Greece.webp" alt='Greece map'>
